# Car Price Prediction & Classification 🚗📈

This project performs a comprehensive analysis of car prices using Machine Learning techniques. It includes both a **Regression** task to predict selling prices and a **Classification** task to categorize vehicles into price segments.

## 📋 Overview
The dataset contains over 72,000 car records with features such as mileage, year, engine size, and fuel type. The goal is to provide accurate valuation and categorization of used vehicles.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
- **Environment:** Jupyter Notebook / Google Colab

## 🚀 Tasks Implemented
### 1. Exploratory Data Analysis (EDA)
- Detailed visualization of feature correlations.
- Analysis of price distributions and key value drivers.

### 2. Regression (Linear Regression)
- Predicted the continuous value of car prices.
- Achieved high accuracy through careful feature engineering and scaling.

### 3. Classification (KNN)
- Categorized cars into three buckets: **Cheap, Moderate, and Expensive**.
- Optimized hyperparameters using **GridSearchCV**.

## 📊 Results
- Identified **Year** and **Mileage** as the most significant factors affecting car prices.
- The KNN classifier successfully categorized vehicles with high precision.

## 📂 Project Structure
- `ML_Assignment_1_Car_Price.ipynb`: Main Jupyter Notebook with code and analysis.
- `car_price.csv`: Dataset used for training and testing.
